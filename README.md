# Autoparts API

### REST API for inventory control of an auto parts store made with dotnet.

### ➦ Technologies Used:

- .NET / C# - SDK Version 10.0
- PostgreSQL
- Docker
- Git
- Github

### ➦ Running the Application:

In your terminal (bash/powershell/cmd), type the following commands:

You will need to have Git installed on your machine.

```
git clone https://github.com/devdouglasa/autoparts-dotnet-api.git
```

Enter the project folder:
```
cd autoparts-dotnet-api
```
You will need to have Docker and Docker Compose installed on your machine.
```
docker compose up --build
```

That's it! Your application will start building and will be available at: http://localhost:5221/ <br>
